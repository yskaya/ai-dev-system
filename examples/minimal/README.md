# Minimal workflow example

Reference walkthrough for **ai-dev-system** artifact shapes. These files show what a small feature looks like after running slash commands in a real project — not generated by the build, but written to demonstrate the expected output.

**Work id:** `001` · **Feature:** bookmark save/list API for a personal dashboard

## How to use this example

1. Install ai-dev-system into your project ([README — Quickstart](../../README.md#quickstart-use-in-a-project)).
2. Open the files below side-by-side with [GUIDE.md](../../GUIDE.md) to see how brief → design → plan → review connect.
3. In your own repo, run `/create-brief`, `/design-web`, `/plan-work`, etc. — the agent fills the same schema headings.
